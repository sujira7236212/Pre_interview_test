# Pre_interview_test

## Description
this project is to merge 3 collection into one merged collection with ascending order.
given
     collection_1, collection_2 already sorted from min(0) to max
     collection_3  already sorted from max to min(0)

## test unit

there are 5 test case in test unit
1. Handles merges three sorted arrays correctly
2. Handles handles empty arrays
3. Handles one empty arrays
4. Handles two empty arrays

## how to set up

## Install dependencies
npm install

## Run unit tests
npm test

## Build project
npm run build
